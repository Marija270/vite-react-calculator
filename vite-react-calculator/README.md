# vite-react-calculator

Jednostavna web aplikacija napravljena u tehnologijama React, Vite i TypeScript. Aplikacija predstavlja mini kalkulator koji podrzava sabiranje, oduzimanje, mnozenje i deljenje kroz dva input polja i prikaz rezultata.

## Funkcionalnosti

- unos dva broja
- operacije sabiranja, oduzimanja, mnozenja i deljenja
- validacija praznih polja i zastita od deljenja nulom
- prikaz rezultata i statusne poruke

## Live Demo

Dodaj svoj Vercel link ovde nakon deploy-a:

`https://your-vercel-project-url.vercel.app`

## Lokalno pokretanje

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

1. Postavi projekat na javni GitHub repozitorijum pod nazivom `vite-react-calculator`.
2. Uloguj se na Vercel i izaberi opciju `Add New Project`.
3. Povezi GitHub nalog i izaberi repozitorijum `vite-react-calculator`.
4. Potvrdi podrazumevana Vite podesavanja i pokreni deploy.
5. Nakon prvog deploy-a, svaki `push` na `main` automatski azurira live verziju aplikacije.
